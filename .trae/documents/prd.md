## 1. 产品概述

匿名意见收集平台是为企业内部打造的匿名反馈系统，旨在让员工在完全匿名的前提下自由表达真实想法与建议，帮助管理层了解组织真实声音，推动持续改进。
- 解决员工因担心身份暴露而不敢说真话的痛点，建立安全的意见通道
- 目标用户为企业内部员工与管理层/HR，核心价值在于提升组织透明度与信任度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员（管理层/HR） | 管理员邀请创建 | 创建调查、查看意见、公开回复、审核内容、查看统计报告、管理标签 |
| 员工 | 管理员创建账号 | 参与调查、提交匿名意见、查看意见广场 |

### 2.2 功能模块

1. **调查管理页**：创建/编辑调查、设置题型、发布调查、管理调查列表
2. **意见提交页**：匿名填写调查、选择标签、提交意见
3. **意见广场页**：浏览所有已公开的意见与官方回复、按标签筛选、搜索
4. **内容审核页**：查看被关键词过滤拦截的提交、人工审核通过或拒绝
5. **统计报告页**：季度数据概览、问题类型趋势图、高频标签排名

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

**匿名提交流程**：员工登录 → 选择进行中调查 → 匿名填写意见（自由文本/评分） → 选择主题标签 → 提交 → 系统关键词检测 → 通过则进入意见广场 / 拦截则进入人工审核 → 管理员审核 → 通过则公开 / 拒绝则屏蔽

**管理员回复流程**：管理员查看意见广场 → 选择意见 → 撰写公开回复 → 回复展示在意见下方（类似问答）

**季度报告流程**：系统按季度自动汇总 → 统计各类型问题数量 → 生成趋势图表 → 管理员查看报告

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深靛蓝 (#1a1a2e) + 暖琥珀色 (#f0a500) 作为强调色，传达专业信任与温暖开放
- 辅助色：柔和灰蓝 (#16213e) 作为背景层次，薄荷绿 (#0f9b58) 表示安全/匿名状态
- 按钮风格：圆角矩形（8px），轻微投影，悬停时微微上浮
- 字体：思源黑体 Noto Sans SC 为主，数据展示用 JetBrains Mono
- 布局风格：左侧导航栏 + 右侧内容区，卡片式内容组织
- 图标风格：线性图标，2px 描边，圆角端点

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 登录页 | 登录表单 | 居中卡片，深色背景，输入框带图标，登录按钮为琥珀色渐变，背景有微妙的几何纹理 |
| 调查管理页 | 调查列表 | 顶部统计卡片（进行中/已结束数量），表格列表，状态标签（绿色进行中/灰色已结束） |
| 调查管理页 | 创建调查表单 | 侧边抽屉式表单，题型拖拽排序，评分题显示星级预览 |
| 意见提交页 | 调查详情 | 卡片式题目列表，自由填写为大文本框，评分题为星级/滑块，底部标签选择器，匿名盾牌图标 |
| 意见广场页 | 意见列表 | 瀑布流卡片，左侧标签色条，卡片含内容摘要+标签+时间+回复标记，悬停微上浮 |
| 意见广场页 | 标签筛选栏 | 左侧垂直标签列表，带数量气泡，选中高亮 |
| 内容审核页 | 待审核列表 | 卡片列表，命中关键词红色高亮，通过/拒绝按钮分别为绿色/红色 |
| 统计报告页 | 季度概览 | 顶部4个统计数字卡片，带环比箭头指示，下方趋势折线图+饼图 |

### 4.3 响应式设计

- 桌面优先设计，主要内容区最大宽度 1280px
- 平板适配：导航栏收缩为汉堡菜单，卡片从三列变两列
- 移动端适配：单列布局，底部标签导航，表格改为卡片列表
